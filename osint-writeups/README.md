## Tools & techniques

### Information gathering
- **Google Dorks** — advanced queries (`site:`, `inurl:`, `filetype:`, etc.)
- **Wayback Machine** — web archives, deleted pages
- **WHOIS / DNS lookup** — domain and infrastructure identification

### Geolocation (GeoINT)
- **Google Maps / Street View** — ground truth verification
- **SunCalc** — time estimation from shadows/sun position
- **Overpass Turbo** — advanced OpenStreetMap queries

### Social media (SOCMINT)
- **Sherlock** — cross-platform username search
- **Maigret** — account profiling

### Images (IMINT)
- **Google Lens / TinEye** — reverse image search
- **ExifTool** — EXIF metadata extraction
- **FotoForensics** — image manipulation analysis

### Frameworks
- **OSINT Framework** (osintframework.com)
- **IntelTechniques** — investigation workflows

## Methodology

1. **Scope** — define the objective and boundaries
2. **Passive collection** — open sources, no direct interaction
3. **Analysis & correlation** — cross-reference collected data
4. **Verification** — confirm through independent sources
5. **Documentation** — trace every step
